Phalcon est un framework rapide et puissant.


	1. Lancer l'installation du Framework --> https://phalconphp.com/en/download/windows
	2. Il suffit de suivre faire la command dans votre terminal.

Une fois l'installation faite nous allons travailler avec WAMP.

	1. Téléchargement de wamp (pour windows) --> https://phalconphp.com/en/download/windows
	2. Il suffit de mettre le dossier "Phalcon" dans le www. de WAMP.
